Base 64 Encoding

Base 64 Base 32

Binary data into text string using a set of 64 different ASCII characters.

Character set -
ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789+/

Padding character "=" is used to ensure that the length of the encoded output is a multiple of 4.

Encoding

Convert input data to binary. (ASCII Byte values) Combine the binary values into a continuous string of binary digits. Divide into 6 bit groups. Convert each group to corresponding decimal value. Map each decimal value to a base 64 character.

Handling Padding

If input is not a multiple of 3, padding is added to the byte representation. To make output length a multiple of 4, add padding to the output.

Decoding

Remove padding. Map base 64 to 6 bit values. combine 6 bit values into 8 bit Bytes.
